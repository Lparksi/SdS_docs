# 基本配置

# 更名

将 configDefault.py 更名为 config.py 将 pluginsConfigDefault.py 更名为 pluginsConfig.py

# 找到3个配置文件

打开星之子的配置文件config.py , pluginsConfig.py 与 go-cqhttp的配置文件config.json

星之子配置

PORT = 5867 # go-cqhttp端口

# 进行配置

将所有的主机(HOST)配置为 127.0.0.1
反向ws与HTTP不要使用一个端口 下面给出可用的配置：
主机(HOST):127.0.0.1 反向ws端口：5867 HTTP端口：5700

星之子配置(config.py)








 
 

from nonebot.default_config import *

SUPERUSERS = {} # 超级管理员
ADMINS = {} # 管理员
OWNER = {} # 所有者

COMMAND_START = {'.', '/', '／'} # 命令前置符

HOST = "0.0.0.0" # go-cqhttp主机
PORT = 5867 # go-cqhttp端口
星之子插件配置(pluginsConfig.py)


 
 





# HTTP Config
# HTTP 配置
HTTP_HOST = "127.0.0.1"
HTTP_PORT = 5700

# 和风天气
# heweather.com

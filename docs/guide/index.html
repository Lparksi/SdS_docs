# How to use it?

不必担心

星之子非常容易配置。

Let's go!

# 下载&配置go-cqhttp

我们选择 go-cqhttp 作为后端，它将为我们提供优越的性能与优秀的体验。

  1. 前往Releases 页面下载相对于的二进制包。
  2. 请参看官方的安装说明
  3. 官方的 配置文件结构,请将反向WS与HTTP客户端打开，星之子需要使用~

# 下载&配置星之子

这是星之子的主程序，星之子的本源，

  1. 前往Releases 页面下载最新版的源码。
  2. 下载&安装对应平台的Python(3.7+)
可能会遇到的问题

遇到无法正常安装nonebot库请更新pip3并切换至国内清华镜像

pip install pip -U
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

更多信息参见issues#1

  1. 在终端(bash,cmd,PowerShell等)执行如下命令
pip3 install -r requirements.txt
  1. 执行启动命令 | 星之本源
python3 bot.py

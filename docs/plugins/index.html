# info

# weather

采用 和风天气 作为数据源，需要配制key。 .weather

# Authority

权限系统。 获取权限：在管理员列表中的用户,向机器人发送' .sudo或.以神之名 即可获得限时的对应的权限。 提供方法：

  • plugins.Authority.checkSupers 检查超级管理员权限，返回值为布尔型。
  • plugins.Authority.checkAdmins 检查管理员权限，返回值为布尔型。 .sudo .以神之名

# Restart

重启服务端(go-cqhttp)，需要权限：最低管理员 .restart
